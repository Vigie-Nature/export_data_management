# export_data_management

Ce projet vise à importer, filtrer et standardiser les exports "à plat" des données des différents observatoires de Vigie-Nature pour ensuite les uploader vers l'espace serveur du MESU

Les exports sont déposés sur un serveur et doivent permettre la mise à disposition des données et la production de bilans et d'indicateurs

La documentation est disponible ici : [Catalogue de données Vigie-Nature](https://vigie-nature.github.io/data_catalogue_vn/)
